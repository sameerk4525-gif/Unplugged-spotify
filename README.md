# 🎵 Unplugged Spotify

A lightweight **C-based music playlist manager** that simulates core features of a music app like Spotify — built entirely using **data structures and file handling**.
Features

*  Create and manage playlists
*  Add songs to playlists
*  Remove songs
*  Store and retrieve playlist data from files
*  Navigate through songs efficiently

Tech Stack

* **Language:** C
* **Concepts Used:**

  * Data Structures (Linked Lists / Arrays)
  * File Handling
  * Modular Programming
